Based on the primary \nkind of data used in the mining process, Web mining tasks are categorized \ninto three main types: <i>Web structure mining</i>, <i>Web content mining</i> \nand <i>Web usage mining</i>. This book consists of two parts. The first part covers the data mining and machine learning foundations, where all the essential algorithms of data mining and machine learning are presented. The second part covers the key topics of Web mining, where <i><b>Web crawling</b></i>, <i><b>search</b></i>, <i><b>social network analysis</b></i>, <i><b>structured data extraction</b></i>, <i><b>information integration</b></i>, <i><b>opinion mining and sentiment analysis</b></i>, <i><b>Web usage mining</b></i>, <i><b>query log mining</b></i>, <i><b>computational advertising</b></i>, and <i><b>recommender systems</b></i> are all treated in breadth and in depth (the SVD matrix factorization algorithm of Simon Funk used in Netflix Prize Contest is described in detail).\n</ul>\n<p><b>What is new in the second edition?</b> Most chapters have been updated. The major changes are in Chapter 11 and Chapter 12, which have been re-written and significantly expanded. When the first edition was written, opinion mining (Chapter 11) was still in its infancy. Since then, the research community has proposed many novel techniques to solve various aspects of the problem. To include the latest developments for the Web usage mining chapter (Chapter 12), the topics of recommender systems and collaborative filtering, query log mining, and computational advertising have been added. This new edition is thus considerably longer, from a total of 532 pages in the first edition to a total of 622 pages in this second edition. \n<p> <b>Teaching and Learning</b>: Although the book is titled "Web Data \nMining", it also covers the key topics of data mining, information retrieval, \nand text mining. Thus, it is suitable for a data mining course, \nin which the students learn not only data mining, but also Web mining and text mining. The book is appropriate for advanced undergraduate students, graduate students, researchers and practioners in the field. \nNo prior knowledge of data mining or machine learning is assumed. \n\n<br>\n<h3> Order the Second Edition </h3>\n<TABLE  BORDER=0 width="100%">\n<tr>\n<td width="49%" valign=top>\n<ul>\n<li>From <a href="http://www.amazon.com/Web-Data-Mining-Data-Centric-Applications/dp/3642194591/ref=sr_1_3?s=books&ie=UTF8&qid=1302988351&sr=1-3"> Amazon.com</a>\n<li>From <a href="http://www.springer.com/computer/database+management+%26+information+retrieval/book/978-3-642-19459-7">Springer</a>\n<li>Online version available from <a href="http://www.springerlink.com/content/978-3-642-19460-3#section=912389&page=1&locus=0">Springer</a>\n</ul>\n<td> <td width="2%">\n<td width="49%" valign=top>\n2011, 622 p., Hardcover <br>\nISBN 978-3-642-19459-7\n</tr>\n</table>\n\n<h3> Order the First Edition </h3>\n<TABLE  BORDER=0 width="100%">\n<tr>\n<td width="49%" valign=top>\n<ul>\n<li> From <a href="http://www.amazon.com/gp/product/3540378812?ie=UTF8&tag=univofilliatc-20&linkCode=as2&camp=1789&creative=9325&creativeASIN=3540378812">Amazon.com</a><img src="http://www.assoc-amazon.com/e/ir?t=univofilliatc-20&l=as2&o=1&a=3540378812" width="1" height="1" border="0" alt="" style="border:none !important;
